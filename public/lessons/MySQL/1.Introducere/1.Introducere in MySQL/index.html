Introducere în MySQL.

Ce este MySQL. Instalare server.

MySQL reprezintă un sistem de gestionare a bazelor de date relaționale (SGBD). În prezent, este unul dintre cele mai populare sisteme de gestionare a bazelor de date.

Dezvoltatorul inițial al acestui SGBD a fost compania suedeză MySQL AB. În 1995, au lansat prima versiune MySQL. În 2008, compania MySQL AB a fost achiziționată de către Sun Microsystems, iar în 2010, Oracle a preluat Sun, dobândind astfel drepturile asupra mărcii MySQL. Prin urmare, MySQL este în prezent dezvoltat sub egida Oracle.

Versiunea curentă și relevantă a SGBD este versiunea 8.0, lansată în ianuarie 2018, dar pentru aceasta sunt eliberate constant subversiuni.

MySQL este caracterizat de compatibilitatea cu diferite platforme, având distribuții pentru diverse sisteme de operare, inclusiv cele mai populare versiuni de Linux, Windows și MacOS.

Site-ul oficial al proiectului: https://www.mysql.com/.
