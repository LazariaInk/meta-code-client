Obținerea unui interval de rânduri.

Operatorul LIMIT.

Operatorul LIMIT permite extragerea unui număr specific de rânduri și are următoarea sintaxă:

LIMIT [offset,] rowcount

Dacă operatorului LIMIT i se transmite un singur parametru, acesta indică numărul de rânduri de extras. Dacă sunt transmiși doi parametri, primul parametru stabilește offset-ul față de început, adică câte rânduri să se sara, iar al doilea parametru indică și numărul de rânduri de extras.

De exemplu, să selectăm primele trei rânduri:

SELECT * FROM Products
LIMIT 3;

Acum vom utiliza al doilea parametru și vom specifica offsetul de la care trebuie să înceapă extragerea:

SELECT * FROM Products
LIMIT 2, 3;

În acest caz, se sare peste primele două rânduri și se extrag următoarele 3 rânduri:

De obicei, operatorul LIMIT este folosit împreună cu operatorul ORDER BY:

SELECT * FROM Products
ORDER BY ProductName
LIMIT 2, 3;
