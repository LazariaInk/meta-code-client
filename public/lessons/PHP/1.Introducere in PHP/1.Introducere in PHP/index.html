Prezentare generală a limbajului de programare PHP.

La momentul actual PHP este unul dintre cele mai răspândite limbaje de programare web. Majoritatea covârșitoare a site-urilor și serviciilor web de pe internet sunt scrise folosind PHP. Conform unor estimări PHP este utilizat pe mai mult de 80% din site-uri. Și această popularitate nu este surprinzătoare. Pragul scăzut de intrare și simplitatea limbajului permit crearea rapidă și ușoară a site-urilor și portalurilor de diverse complexități. Iar răspândirea largă asigură cererea ridicată.

PHP a fost creat în 1994 de programatorul danez Rasmus Lerdorf și inițial reprezenta un set de scripturi pe alt limbaj, Perl. Ulterior acest set de scripturi a fost rescris într-un interpretator pe limbajul C. Și de la începutul existenței sale PHP (abreviere de la PHP: Hypertext Preprocessor - PHP: Preprocesor de hipertext) a reprezentat un set convenabil de instrumente pentru crearea simplificată a site-urilor și aplicațiilor web.
