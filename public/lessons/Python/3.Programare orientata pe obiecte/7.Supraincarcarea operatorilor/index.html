Pentru a defini un operator pentru o clasă, aceasta trebuie să implementeze funcția corespunzătoare. De exemplu, pentru a defini operatorul de adunare, se folosește funcția __add__(), astfel că aceasta trebuie definită în cadrul clasei. De exemplu:

class Counter:
   def __init__(self, value):
       self.value = value

   # suprascrierea operatorului de adunare
   def __add__(self, other):
       return Counter(self.value + other.value)

counter1 = Counter(5)
counter2 = Counter(15)
counter3 = counter1 + counter2
print(counter3.value)       # 20

Aici este definită clasa Counter, care are atributul value - un număr convențional. Cu ajutorul funcției __add__, definim operatorul de adunare pentru tipul Counter. Să presupunem că dorim să adunăm un obiect Counter cu un alt obiect Counter. În acest caz, al doilea parametru al funcției va reprezenta celălalt obiect Counter:

def __add__(self, other):
   return Counter(self.value + other.value)

Ca rezultat, returnăm un nou obiect Counter, în care se află suma atributelor value ale ambelor obiecte.

După aceasta, putem aduna două obiecte Counter, iar rezultatul adunării va fi un nou obiect Counter.

În acest caz, a fost implementată doar una dintre posibilele variante ale operatorului de adunare. În exemplul de mai sus, al doilea parametru al funcției reprezintă un alt obiect Counter. Însă, în realitate, acesta poate fi orice tip. De exemplu, dacă dorim să adunăm Counter nu cu un alt obiect Counter, ci cu un număr. Atunci putem defini următorul operator:

class Counter:
   def __init__(self, value):
       self.value = value

   def __add__(self, other):
       return Counter(self.value + other)

counter1 = Counter(5)
counter3 = counter1 + 6
print(counter3.value)       # 11

Aici, operatorul de adunare returnează în continuare un obiect Counter, doar că acum al doilea parametru reprezintă un număr obișnuit.

Tipul returnat de o serie de operatori nu este strict determinat. De exemplu, putem returna și un număr obișnuit:

class Counter:
   def __init__(self, value):
       self.value = value

   def __add__(self, other):
       return self.value + other

counter1 = Counter(5)
result = counter1 + 7
print(result)       # 12

Să analizăm și alte exemple de definire a operatorilor.

Adevărul unui obiect .

Definirea funcției __bool__ permite stabilirea adevărului unui obiect sau, practic, transformarea acestuia în valorile True/False. De exemplu:

class Counter:
   def __init__(self, value):
       self.value = value

   def __bool__(self):
       return self.value > 0

def test(counter):
   if counter:
       print("Counter = True")
   else:
       print("Counter = False")

counter1 = Counter(3)
test(counter1)              # Counter = True

counter2 = Counter(-3)
test(counter2)              # Counter = False

În acest caz, considerăm că, dacă valoarea value din Counter este mai mică de 1, atunci obiectul Counter va fi considerat False, iar la valori pozitive - True. Datorită acestui lucru, putem folosi obiectul Counter în construcții condiționale sau ciclice. De exemplu, în cazul de mai sus, pentru testare este definită funcția test, care în construcția if..else verifică valoarea obiectului Counter și, în funcție de rezultat, afișează un mesaj specific pe consolă.

De exemplu, putem folosi obiectul Counter într-un ciclu while ca și condiție:

class Counter:
   def __init__(self, value):
       self.value = value

   def __bool__(self):
       return self.value > 0

counter1 = Counter(3)

while(counter1):
   print("Counter1:", counter1.value)
   counter1.value -= 1

În acest caz, ciclul while va rula atât timp cât counter1 corespunde valorii True (practic, cât timp valoarea value este mai mare de 0).

Operatori care returnează valoarea bool .

O serie de operații sunt destinate să returneze o valoare logică True sau False. De exemplu, operațiile de comparație:

class Counter:
   def __init__(self, value):
       self.value = value

   def __gt__(self, other):
       return self.value > other.value

   def __lt__(self, other):
       return self.value < other.value

counter1 = Counter(1)
counter2 = Counter(2)

if counter1 > counter2:
   print("counter1 este mai mare decât counter2")
elif counter1 < counter2:
   print("counter1 este mai mic decât counter2")
else:
   print("counter1 și counter2 sunt egale")

Aici, în clasa Counter, sunt definiți operatorii < (funcția __lt__()) și > (funcția __gt__()). În acest caz, comparăm cu un alt obiect Counter. În realitate, comparăm valorile atributelor celor două obiecte.

def __gt__(self, other):
   return self.value > other.value

def __lt__(self, other):
   return self.value < other.value

Ulterior, putem aplica operațiile corespunzătoare celor două obiecte Counter:

if counter1 > counter2:

Operații de accesare prin index.

 

O serie de operatori permit accesarea obiectului prin index, folosind paranteze pătrate:

obj[index]

De obicei, astfel de operații sunt aplicate colecțiilor, care vor fi discutate în articolele următoare. De exemplu, astfel de operații pot fi utilizate pentru a obține sau modifica un element al unei liste de valori. În realitate, aceste operații pot fi aplicate oricărui obiect, iar indexul folosit poate reprezenta orice. Să analizăm operațiile de accesare prin index pe exemplul obținerii unei valori prin index:

class Person:
   def __init__(self, name, age):
       self.__name = name
       self.__age = age

   def __getitem__(self, prop):
       if prop == "name":
           return self.__name
       elif prop == "age":
           return self.__age
       return None

tom = Person("Tom", 39)

print("Name:", tom["name"])  # Name: Tom
print("Age:", tom["age"])  # Age: 39
print("Id:", tom["id"])  # Id: None

Așadar, aici este definită clasa Person, care conține două câmpuri private - __name și __age. Pentru a implementa obținerea datelor prin index, este definită funcția __getitem__(). Ca al doilea parametru, această funcție primește valoarea care îndeplinește rolul de index. În cazul nostru, acesta va fi numele atributului. În funcție de valoarea transmisă, returnăm fie valoarea atributului __name, fie valoarea atributului __age. Dacă este transmis un nume de atribut nevalid, returnăm None.

Pentru a obține valoarea prin index, transmitem indexul - numele atributului - în paranteze pătrate:

tom["name"]

Verificarea existenței unei proprietăți.

Operatorul in permite verificarea existenței unei anumite valori într-o secvență - un set de valori:

valoare in secvență

Dacă valoarea este prezentă în secvență, se returnează True, altfel se returnează False. De exemplu, să verificăm existența unei proprietăți într-un obiect:

class Person:
   def __init__(self, name, age):
       self.name = name
       self.age = age

   def __contains__(self, prop):
       if prop == "name" or prop == "age":
           return True
       return False

tom = Person("Tom", 39)
print("name" in tom)  # True
print("id" in tom)  # False

Funcția __contains__() este responsabilă de implementarea operatorului in. Ca prim parametru, așa cum este obișnuit, se indică obiectul curent - obiectul aflat la dreapta operatorului in. Iar ca al doilea parametru - valoarea verificată - aceasta este indicată la stânga operatorului in. În acest caz, dacă al doilea parametru este egal cu "name" sau "age", returnăm True. Aceasta înseamnă că atributul există în obiect.

Prin urmare, expresia "name" in tom va returna True, iar expresia "id" in tom va returna False.

Implementarea operatorilor în perechi.

Unii operatori, cum ar fi operatorii de comparație, sunt mai convenabil de implementat în perechi. Dacă implementăm operatorul ==, putem implementa imediat și operatorul !=. În loc să scriem aceeași logică de două ori, putem implementa un operator prin intermediul celuilalt:

class Counter:
   def __init__(self, value):
       self.value = value

   def __eq__(self, other):
       return self.value == other.value

   def __ne__(self, other):
       return not (self == other)

   def __gt__(self, other):
       return self.value > other.value

   def __le__(self, other):
       return not (self > other)

   def __lt__(self, other):
       return self.value < other.value

   def __ge__(self, other):
       return not (self < other)

c1 = Counter(1)
c2 = Counter(2)

print(c1 == c2)  # False
print(c1 != c2)  # True

print(c1 < c2)  # True
print(c1 >= c2)  # False

În acest caz, operatorul != returnează inversarea rezultatului operatorului ==, care este definit mai sus:

def __eq__(self, other):
   return self.value == other.value

def __ne__(self, other):
   return not (self == other)

Similar, sunt definiți operatorii < și >=, precum și > și <=.
