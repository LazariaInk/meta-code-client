Frame-uri.

Frame-urile permit încorporarea unei alte pagini web într-o pagină web. Frame-urile sunt reprezentate de elementul iframe. Să presupunem că dorim să încorporăm pe o pagină web pagina de start a Wikipedia:

<!DOCTYPE html>
<html>
   <head>
       <meta charset="utf-8">
       <title>Frame-uri în HTML5</title>
   </head>
   <body>
       <h2>Elementul iframe</h2>
       <iframe src="http://wikipedia.com" width="400" height="200">
       </iframe>
   </body>
</html>

Elementul iframe nu conține în sine niciun conținut. Toate setările lui se fac prin atribute:

- src: stabilește calea completă către resursa încărcată

- width: lățimea frame-ului

- height: înălțimea frame-ului

Trebuie de menționat că nu toate site-urile pot fi deschise în frame-uri, deoarece pe partea serverului web pot exista restricții privind deschiderea în frame-uri.
